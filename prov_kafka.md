#### Install

``$ sudo useradd kafka -m``

``$ sudo passwd kafka``

``$ sudo adduser kafka sudo``

``$ mkdir ~/Downloads``

``$ curl "https://www.apache.org/dist/kafka/2.3.1/kafka_2.12-2.3.1.tgz" -o ~/Downloads/kafka.tgz``

``$ mkdir ~/kafka && cd ~/kafka``

``$ tar -xvzf ~/Downloads/kafka.tgz --strip 1``
